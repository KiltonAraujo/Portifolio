Diretoria de TI

Deploy Github

Resumo

O Portal Institucional da Diretoria de TI do Campus IFRN Natal Central é um projeto que visa facilitar a comunicação com o público e atender às necessidades tecnológicas dos docentes do campus. Ele oferece tutoriais simples e didáticos, promovendo um ambiente acadêmico mais eficiente e solucionando dúvidas relacionadas ao uso de ferramentas tecnológicas essenciais.

Tecnologias

  1. HTML5
  2. CSS3
  3. JavaScript

Garçom

Deploy Github

Resumo

Garçom é uma plataforma full stack que funciona como um cardápio digital dinâmico, permitindo que restaurantes atualizem informações, preços e disponibilidade de pratos em tempo real. O sistema torna o atendimento mais rápido, reduzindo erros de comunicação e trazendo uma solução acessível para digitalização de pequenos e médios estabelecimentos.

Tecnologias

  1. React
  2. Typescript
  3. Tailwind CSS
  1. Node JS
  2. PostgreSQL
  3. Vercel

Iformandos

Deploy Github

Resumo

IFormandos é uma rede social voltada para egressos do IFRN Campus Natal Central, permitindo que alunos mantenham contato após a formatura. A plataforma digitaliza placas de formatura, reduzindo o acúmulo físico nas escolas, e oferece um espaço centralizado onde turmas podem registrar fotos, histórias e interações, preservando vínculos e facilitando o acesso às recordações acadêmicas.

Tecnologias

  1. HTML
  2. CSS
  3. Python
  1. JavaScript
  2. Django
  3. MySQL
