Diretoria de TI

Portal Institucional da Diretoria de TI do Campus IFRN Natal Central

Deploy Github

  1. Tecnologias

    1. HTML5
    2. CSS3
    3. JavaScript

  2. Problema

    A comunidade escolar tinha dificuldade de encontrar tutoriais, informações e suporte da equipe de TI, causando atrasos e dependência do atendimento presencial.

  3. Papel

    Frontend Developer • UX/UI Designer

    Diferencial

    Sistema de busca inteligente sem backend.

  4. Solução

    Criei um site leve e responsivo para é facilitar a comunicação com o público e atender às necessidades tecnológicas dos docentes. Ele oferece tutoriais simples e didáticos, promovendo um ambiente acadêmico mais eficiente e solucionando dúvidas relacionadas ao uso de ferramentas tecnológicas essenciais.

Garçom

Plataforma para Cardápio Digital com atualizações em tempo real

Deploy Github

  1. Tecnologias

    1. React
    2. Typescript
    3. TailwindCSS
    1. Node JS
    2. PostgreSQL
    3. Vercel

  2. Problema

    Restaurantes que desejam digitalizar seus cardápios enfrentam soluções caras, difíceis de integrar e sem atualização em tempo real. A falta de um sistema acessível e personalizável impede que eles ofereçam ao cliente uma experiência moderna e dinâmica.

  3. Papel

    Fullstack Developer • UX/UI Designer

    Diferencial

    Status do pedido em tempo real, Arquitetura escalável e design moderno.

  4. Solução

    Participei do desenvolvimento de uma plataforma full stack que funciona como um cardápio digital dinâmico, permitindo que restaurantes atualizem informações, preços e disponibilidade de pratos em tempo real. O sistema torna o atendimento mais rápido, reduzindo erros de comunicação e trazendo uma solução acessível para digitalização de pequenos e médios estabelecimentos.

Iformandos

Rede Social para Egressos com Memórias Digitais e Placas Virtuais

Deploy Github

  1. Tecnologias

    1. HTML
    2. CSS
    3. Python
    1. JavaScript
    2. Django
    3. MySQL

  2. Problema

    Escolas acumulam placas físicas de formatura, gastando espaço e dinheiro, e os alunos perdem o contato após a conclusão.

  3. Papel

    Fullstack Developer • UX/UI Designer

    Diferencial

    Solução moderna para um problema comum em instituições de ensino.

  4. Solução

    Participei do desenvolvimento de uma rede social intuitiva voltada para egressos, permitindo que alunos mantenham contato após a formatura. A plataforma digitaliza placas de formatura, reduzindo o acúmulo físico nas escolas, e oferece um espaço centralizado onde turmas podem registrar fotos, histórias e interações, preservando vínculos e facilitando o acesso às recordações acadêmicas.
